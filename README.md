# Bootstrap 5 Responsive Website

A fully responsive, mobile-first website built using Bootstrap 5, inspired by the Net Ninja Bootstrap 5 tutorial series. This project showcases how to effectively use Bootstrap’s layout utilities, grid system, components, and responsive classes to build a clean and modern webpage.



## Technologies Used

* HTML5
* CSS3
* Bootstrap 5



## Features

### Responsive Layout

* Built with Bootstrap's mobile-first approach.
* Uses the grid system, responsive breakpoints, and utility classes for a flexible design.



### Navigation Bar

* Fully responsive navbar with a collapsible menu on smaller devices.
* Smooth layout adjustments at different screen sizes.



### Bootstrap Components

* Stylish buttons, cards, and forms using Bootstrap’s built-in classes.
* Reusable UI elements for quick prototyping or larger projects.



### Clean \& Modern Design

* Consistent spacing, typography, and color utilities.
* Simple structure that’s easy to extend or customize.
